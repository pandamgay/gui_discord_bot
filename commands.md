# 명령어
> 관리자 명령어는 제목에 ★이 표시됩니다.



## ★ 현재인원-새로고침
*서버 내 현재 인원 채널의 이름을 변경합니다.*  

### 기능
서버 내 현재 인원 채널의 이름을 "현재 인원: 00명 📡"의 형식으로 변경합니다.

### 용도
위 동작은 멤버가 서버에 참여했을 때와 멤버가 서버에서 나갔을 때 자동으로 실행됩니다.  
따라서, 위 명령어는 모종의 이유로 현재 인원의 채널명이 새로고침되지 않았을 때
 수동으로 새로고침하기 위한 명령어입니다.

### 커스텀 설정
없음


### 참고/사용법
- 위 명령어를 사용하기 위해서는 현재 인원 채널이 서버에 존재해야 합니다.
- 현재 인원 채널은 음성 채널이여야 하며, 비공개 음성 채널을 권장합니다.
- 위 명령어는 "현재 인원 이벤트"를 활성화한 후에만 사용할 수 있습니다.
