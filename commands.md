## ★ /경고-부여
*멤버에게 경고를 부여합니다.*

### 기능
서버 내 멤버에게 경고 역할을 부여하고, 벤 로그를 기록 및 데이터베이스에 저장합니다.

### 용도
서버 내 멤버가 서버 규칙을 위반했을 때 경고를 부여하기 위해 사용합니다.

### 커스텀 설정
- 경고 기간 기본값 변경

### 매개 변수
| 설명                    | 매개변수 | 타입               |
|-----------------------|------|------------------|
| 경고를 부여할 멤버            | 멤버   | `discord.Member` |
| 경고 사유                 | 사유   | `str`            |
| 경고 기간 (기본값: {설정 기간}일) | 기간   | `int = {설정 기간}`  |


### 참고/사용법
- 경고 기간은 일 단위로 설정됩니다.
- 경고는 자정마다 자동으로 만료됩니다.
- 위 명령어는 [경고 유틸리티](#24)를 활성화한 후에만 사용할 수 있습니다.

## ★ /이벤트-생성
*이벤트를 생성합니다.*

### 기능
서버 내에서 진행할 이벤트를 생성하고, 이벤트 메시지를 전송합니다.

### 용도
서버 내에서 이벤트를 진행하기 위해 사용합니다.

### 커스텀 설정
- 이벤트 공지 문구 설정

### 매개 변수
| 설명                   | 매개변수 | 타입         |
|----------------------|------|------------|
| 이벤트 제목을 입력하세요.       | 제목   | `str`      |
| 이벤트에 대한 설명을 입력하세요.   | 설명1  | `str`      |
| 추가 설명을 입력하세요. (선택사항) | 설명2  | `str = ""` |
| 추가 설명을 입력하세요. (선택사항) | 설명3  | `str = ""` |

### 참고/사용법
- 이벤트는 한 서버에 하나씩만 진행할 수 있습니다.
- 위 명령어는 [서버 이벤트 유틸리티](#22)를 활성화한 후에만 사용할 수 있습니다.

## ★ /이벤트-종료

### 용도
서버 내에서 진행 중인 이벤트를 종료하기 위해 사용합니다.

### 커스텀 설정
없음

### 매개 변수
없음

### 참고/사용법
- 위 명령어는 [서버 이벤트 유틸리티](#22)를 활성화한 후에만 사용할 수 있습니다.

> ```

> @이벤트 참여자
> ```

## /경험치-도박
*경험치를 도박합니다.*

### 기능
보유 중인 경험치를 사용해 도박을 진행하며, 결과에 따라 경험치를 획득하거나 잃습니다.

### 용도
서버 내 멤버가 경험치를 소모하여 도박 콘텐츠를 즐길 때 사용합니다.

### 커스텀 설정
- 최댓값, 최솟값 설정

### 매개 변수
| 설명                                                     | 매개변수 | 타입     |
|--------------------------------------------------------|------|--------|
| 배율을 입력하세요. 2배: 1/2, 5배: 1/5                            | 배율   | `str`  |
| 도박할 경험치 양을 입력하세요. 2배: {최솟값} ~ {최댓값}, 5배: {최솟값} ~ {최댓값} | 경험치  | `int`  |

### 참고/사용법
- 위 명령어는 [경험치 유틸리티](#25)를 활성화한 후에만 사용할 수 있습니다.
- 배율에 따라 도박의 최댓값과 최솟값이 다릅니다.
- 🟥🟦🟩🟨🟧중 하나를 선택하여 도박을 진행합니다.
