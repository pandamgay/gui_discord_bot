## ★ /현재인원-새로고침
*현재 인원을 새로고침합니다.*  

### 기능
서버 내 현재 인원 채널의 이름을 "현재 인원: 00명 📡"의 형식으로 변경합니다.

### 용도
위 동작은 멤버가 서버에 참여했을 때와 멤버가 서버에서 나갔을 때 자동으로 실행됩니다.  
따라서, 위 명령어는 모종의 이유로 현재 인원의 채널명이 새로고침되지 않았을 때 
수동으로 새로고침하기 위한 명령어입니다.

### 커스텀 설정
없음

### 매개 변수
없음

### 참고/사용법
- 위 명령어를 사용하기 위해서는 현재 인원 채널이 서버에 존재해야 합니다.
- 현재 인원 채널은 음성 채널이여야 하며, 비공개 음성 채널을 권장합니다.
- 위 명령어는 [현재 인원 이벤트](#20)를 활성화한 후에만 사용할 수 있습니다.

<a name="2"></a>
## ★ /경고-부여
*유저에게 경고를 부여합니다.*

### 기능
서버 내 멤버에게 경고 역할을 부여하고, 벤 로그를 기록 및 데이터베이스에 저장합니다.

### 용도
서버 내 멤버가 서버 규칙을 위반했을 때 경고를 부여하기 위해 사용합니다.

### 커스텀 설정
- 경고 기간 기본값 변경

### 매개 변수
| 설명                    | 매개변수 | 타입               |
|-----------------------|------|------------------|
| 경고를 부여할 유저            | 유저   | `discord.Member` |
| 경고 사유                 | 사유   | `str`            |
| 경고 기간 (기본값: {설정 기간}일) | 기간   | `int = {설정 기간}`  |


### 참고/사용법
- 경고 기간은 일 단위로 설정됩니다.
- 경고는 자정마다 자동으로 만료됩니다.
- 위 명령어는 [경고 유틸리티](#24)를 활성화한 후에만 사용할 수 있습니다.

## ★ /경고-갱신
*경고가 만료된 유저를 갱신합니다.(매일 정오에 작동되므로 가급적 사용하지 마세요.)*

### 기능
서버 내 멤버의 만료된 경고를 갱신하고, 경고 역할을 제거합니다.

### 용도
위 동작은 매일 정오에 자동으로 실행됩니다.  
따라서, 위 명령어는 모종의 이유로 서버 내 멤버의 경고 정보가 새로고침되지 않았을 때 
수동으로 새로고침하기 위한 명령어입니다.

### 커스텀 설정
없음

### 매개 변수
없음

### 참고/사용법
- 위 명령어는 [경고 유틸리티](#24)를 활성화한 후에만 사용할 수 있습니다.

<a name="4"></a>
## ★ /이벤트-생성
*이벤트를 생성합니다.*

### 기능
서버 내에서 진행할 이벤트를 생성하고, 이벤트 메시지를 전송합니다.

### 용도
서버 내에서 이벤트를 진행하기 위해 사용합니다.

### 커스텀 설정
없음

### 매개 변수
| 설명                   | 매개변수 | 타입         |
|----------------------|------|------------|
| 이벤트 제목을 입력하세요.       | 제목   | `str`      |
| 이벤트에 대한 설명을 입력하세요.   | 설명1  | `str`      |
| 추가 설명을 입력하세요. (선택사항) | 설명2  | `str = ""` |
| 추가 설명을 입력하세요. (선택사항) | 설명3  | `str = ""` |

### 참고/사용법
- 이벤트는 한 서버에 하나씩만 진행할 수 있습니다.
- 위 명령어는 [서버 이벤트 유틸리티](#22)를 활성화한 후에만 사용할 수 있습니다.
- 이벤트 메시지는 아래 형식으로 전송됩니다.
> ```
> # 이벤트 안내🎁: {제목}
> {설명1}
> {설명2}
> {설명3}
> 이 메시지에 ✅를 눌러주시면 이벤트 참여가 완료됩니다.
> @everyone
> ```

## ★ /이벤트-종료
*이벤트를 종료합니다.*

### 기능
서버 내 진행 중인 이벤트를 종료하고, 이벤트 결과 메시지를 전송합니다.  
실질적으로 위 명령어가 하는 역할은 이벤트 참여자 역할을 가진 멤버들의 역할을 제거합니다.

### 용도
서버 내에서 진행 중인 이벤트를 종료하기 위해 사용합니다.

### 커스텀 설정
없음

### 매개 변수
없음

### 참고/사용법
- 위 명령어는 [서버 이벤트 유틸리티](#22)를 활성화한 후에만 사용할 수 있습니다.
- 이벤트 종료 메시지는 아래 형식으로 전송됩니다.
> ```
> # 이벤트 종료
> 현재 이벤트가 종료되었습니다. 참여해주신 모든 분들께 감사합니다.
> @이벤트 참여자
> ```
